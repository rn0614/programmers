# HTML 태그 종류

h<i>n</i> : 제목태그

p : 단락태그 (줄바꿈)

br:  줄바꿈태그

hr : 가로선태그

b : 굵게

i : 이탤릭체

span : 영역묶기 (줄바꿈 없음)



ul _li : 순서 없는 목록

ol_li : 순서 있는 목록

table : 테이블 태그

tr_td_th : 열, 행, 제목   ex)  `<tr><td></td><td></td></tr>`

`<td colspan="합칠 셀의 개수">`

`<td row span="합칠 셀의 개수">`

img src="경로" : 그림

script src="경로" : js 연동

a href="링크주소" target="_blank" : 링크 사이트로 이동



form action="자바단 or jsp" method="post":
